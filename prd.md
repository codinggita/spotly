A maximum of 4-5 files are allowed per PR. For example, if you’re adding a toggle button and it involves 4-5 files, that’s fine. However, if you’re implementing a login/sign-up page and end up with 6 files, you should split them into two separate PRs with 4-5 files each. Name them clearly, such as “Implemented login/sign-up page (1)” and “Implemented login/sign-up page (2)”.
One more thing , the page-wise components you are creating is fine ,  but you have to make sure that the particular line of code in each file should not exceed 250 , if it goes beyond 250 lines then you have to create a another sub-components of that main components.
